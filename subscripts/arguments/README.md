### Arguments

| Parameter		    | Description						|
| ---			    | ---							|
| `$#`			    | Number of input parameters				|
| `$0`			    | Name of the script					|
| `$1` -`$9`		| The parameter list elements from 1 to 9.		|
| `${10}` -`${N}`	| The parameter list elements from 10 to N.		|

### Set Arguments
```bash
    set -- "-s 123"
```
